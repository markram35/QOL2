About me

I graduated in 2022 with an undergraduate degree in Business Analytics from the University of Tennesee with a minor in Environmental Studies. During my time at the Universtiy of Tennessee I had the honor of becoming a Global Leadership Scholar and studying abroad in London. Comprehensive Time Series Application

An application developed to be able to create a large variety of time series models based upon time series data submitted by the user. So far this app works on annual, quarterly, monthly, and daily data.

Some of the models which can be developed include Summary models, Basic Time Series Models, Linear Regression Models, Exponential Smoothing models, and both seasonal and non-seasonal ARIMA models. The application also provides model statistics â€“ which can be used for judging the best model â€“ as well as the forecasted values of each model of your choice.

FirstCare Medical Ceneter database

As part of my Undergraduate Database class, we were tasked with creating a fully functioning working database which can generate reports with a fully funcitoning working user interface. The database was first outlined in an ERD developed using Microsoft Visio, and then built in Microsoft SQL server. This database was then imported into Microsoft Access, which is what was used to create the User Interface.

This user interface includes ways to submit new data to every database entity, as well as form request interfaces to make querying the database more efficient and user friendly.

You can download the full report, which includes all the code for the database as well as graphics detailing the user interface here.
